# UI Components

Reusable UI components exported from `@muzammil/ui`.

## Usage

```tsx
import { Button, Card, DataTable, Dialog, Input, Text, Tooltip } from '@muzammil/ui';

export function Example() {
	return <Button>Save</Button>;
}
```

## Includes

- Core UI: button, card, badge, avatar, tabs, tooltip, dropdown
- Forms: input, select, checkbox, switch, textarea, form fields
- Data UI: table/data-table, list, chart, timeline, stepper
- Layout: container, separator, scroll-area, resizable, sidebar
- Overlays: dialog, drawer, sheet, popover, alerts
- Content: typography, media, rich-text

## Notes

- Import components from the package root.
- For advanced usage, see each feature folder README.
