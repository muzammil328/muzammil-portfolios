# Portfolio App

Next.js portfolio application with shared UI components.

## Features


- Next.js 16 with TypeScript
- Tailwind CSS for styling
- Dark/Light theme support
- TanStack Query for data fetching
- React Hook Form with Zod validation
- Sonner for toasts
- Jest testing setup
- Sass support
